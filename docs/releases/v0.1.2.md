# Starsector System Ranker v0.1.2

Solver-quality release. No CLI, config, or data-format changes — existing
commands, saves, and databases work unchanged.

## Improvements

- **Smarter placement of scarce colony upgrades.** Story-point improvements and
  alpha cores are no longer claimed by whichever colony happens to build first.
  The solver now discovers the full set of facilities a plan will build, then
  allocates each scarce upgrade to the facility where it pays off most and
  schedules it as early as that facility allows. This applies to both kinds of
  solve: maximizing income/stability/defense (most benchmarked points improved,
  several by 5–30%) and reaching a threshold fastest (upgrades go to whatever
  reaches the goal soonest).

## Upgrading

Unpack this release and run the installer again, or run `system_solver install`.
No data migration is needed; existing databases are unaffected.

See [v0.1.1](v0.1.1.md) for the previous release and [v0.1.0](v0.1.0.md) for the
initial feature set.
